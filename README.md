# NEVEKO Landingpage

Einfache statische Landingpage für NEVEKO Advisory, gehostet über Azure Static Web Apps.

## Struktur

- `index.html` – Hauptseite
- `styles.css` – zentrales Styling

## Lokale Entwicklung

Einfach die `index.html` im Browser öffnen.

## Deployment (Azure Static Web Apps)

Das Repository mit Azure Static Web Apps verbinden und als App-Artifact das Root-Verzeichnis (`/`) angeben.
